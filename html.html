Formbricks Challenge Dashboard

Step 1: Locally Run the App

Run Formbricks instance locally using Docker/Docker Compose.

python main.py formbricks up Running

Step 2: Stop the App

Gracefully stop and clean up the running Formbricks instance.

python main.py formbricks down Stopped

Step 3: Generate Realistic Data

Programmatically generate survey/user/response data using an LLM.

python main.py formbricks generate Completed

Step 4: Fill the Data using APIs

Seed the Formbricks instance via Management API (surveys/users) and Client API (responses).

python main.py formbricks seed Running

Seeding Overview

  • 5 Surveys with realistic questions
  • At least 1 Response per survey
  • 10 Users with Manager/Owner access
  • API Only, no direct DB edits
